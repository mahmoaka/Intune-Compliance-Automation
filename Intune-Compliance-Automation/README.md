# Intune Compliance & Stale Device Reporting (Mock + Real Version)

This Project demonstrates how Intune compliance reporting works using Powershell. It includes both a **mock version**(safe for Github) and a **real Intune Version**

## What this Project does
- Simulates Intune device data using PowerShell
- Generates a full compliance report(CSV)
- Identifies stale devices (no sync for 30 days)
- Exports stale device report (CSV)
- Show how the script work with **real Intune data** using Microsoft Graph
